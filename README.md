## Обо мне

Я студент 3 курса ТПУ, специализируюсь на системном и сетевом программировании на C++. Основной фокус -- многопоточность, асинхронный ввод-вывод, оптимизация производительности и низкоуровневая работа с памятью.
Принцип в работе: довожу задачи до рабочего состояния, избегая поверхностных решений. При написании кода учитываю требования масштабируемости и долгосрочной поддержки.
Выбрал C++ из-за желания глубокого понимания работы системы -- от управления памятью до сетевых протоколов. Это позволяет создавать эффективные решения там, где высокоуровневые языки не справляются с требованиями по производительности.
